Overview

This is the first newsletter for the Alt Stack Team - a new team at OP Labs dedicated to building and maintaing alternative implementations of the OP Stack.

Alt Stack Team projects include:

  • op-reth
  • kona
  • op-alloy
  • op-rs
  • superchain

If you're interested in joining the Alt Stack Team, reach out to:

  • @clabby
  • @refcell

or just ship a bunch of prs into the above projects, and we might just reach out to you!

Updates

Holocene Hardfork changes are code-complete in kona 🎉.

To unblock ourselves on implementing action tests without the op-node's derivation pipeline being feature complete with Holocene by configuring op-e2e to skip Holocene action tests if the environment variable is set to run op-program, allowing action tests to run if kona is set. This allows us to ship Holocene action tests that run against kona, allowing the op-node folks to activate when feature complete.
